# Data Warehousing

## Projects

### [UC Business Semantics](./dbrx-business-semantics/)

A demo showcasing how to use [UC Business Semantics](https://docs.databricks.com/aws/en/metric-views/) in Databricks to define, manage, and govern business KPIs directly on the platform. Built on top of the [Retail Store Star Schema Dataset](https://www.kaggle.com/datasets/shrinivasv/retail-store-star-schema-dataset?select=fact_sales_denormalized.csv), it demonstrates how embedding your semantic layer in Databricks via metric views provides unified governance through Unity Catalog alongside optimal query performance — reusable across BI tools, SQL clients, and AI agents like Genie.
