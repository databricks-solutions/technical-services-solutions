# Data Warehousing

## Projects

### [Lakebridge Utilities](./lakebridge-utilities/)

Utilities for migrating legacy ETL (SSIS and Informatica) to Databricks using [Lakebridge](https://databrickslabs.github.io/lakebridge) (BladeBridge + Switch LLM transpiler). Includes automation for SSIS packages to Lakeflow Declarative Pipelines and Informatica mappings to Databricks notebooks and jobs, plus sample BladeBridge override configs. Reference only; tune for your environment before production use.
